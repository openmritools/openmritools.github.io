About

The open science MRI community builds incredible tools — but they're scattered across GitHub orgs, lab websites, and word of mouth. Great work gets missed. This directory exists so it doesn't have to.

MRI Tools is a shared overview of open-source software for MRI analysis. If everyone contributes what they know, it becomes a way for the whole community to stay updated and discover each other's work.

How health status works. A weekly GitHub Actions cron job checks each tool's repository. Active means committed within 6 months. Quiet means 6 to 24 months. Inactive means 2+ years or archived. Tools without a GitHub repo show a neutral dot.

How to contribute. Suggest a tool through the form at /contribute, or open a pull request on GitHub. Each tool is a YAML entry — name, URL, one-sentence description, category, language. That's it.

For MR physics and reconstruction tools, see MR-Hub.

Started by Henrik Formo. Open to everyone.
